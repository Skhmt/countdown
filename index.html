Countdown

A desktop countdown timer with global hotkeys

Download - Windows Download - OS X View on GitHub

To install on OS X

  • Get nw.js from the appstore
  • Download the OS X version
  • Place package.json and countdown.html into the same folder as nw.app
  • Run nw.app

To install on Windows

  • Download the Windows version
  • Run countdown.exe

Global hotkeys

Use Ctrl + . to increase time by 5 minutes

Use Ctrl + , to increase time by 1 minute

Use Shift + Ctrl + . to decrease time by 5 minutes

Use Shift + Ctrl + , to decrease time by 1 minute

Use Ctrl + P to pause or play

Hotkeys work globally. As in, they will work even if you're in a game.

Editing

If you want to edit the text or styling on Windows, download nw.js from nwjs.io, download the source, and edit countdown.html. Run it by double-clicking nw.exe.

Directly edit countdown.html to make changes on OS X.
